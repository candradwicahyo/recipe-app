## Recipe App

program ini dibuat dengan menggunakan bahasa pemrograman javascript. program ini berguna untuk mencari sebuah makanan dari penjuru dunia dan juga bisa melihat detail dari makanan tersebut.

* catatan

kalian harus menggunakan bahasa inggris ketika memasukkan value kedalam input pencarian

### dibuat dengan berikut

* HTML
* CSS
  * [Bootstrap](https://getbootstrap.com)
* JavaScript

### lainnya

* [API](https://www.themealdb.com/api.php)
* handphone
* code editor (Acode)
* edit gambar (Pixellab)
